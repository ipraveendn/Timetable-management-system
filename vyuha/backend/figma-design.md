# VYUHA Figma Design Structure

## Main Screens
1. **Login/Registration**
   - Login form with email/password
   - Registration form with college selection
   - "Forgot password" flow

2. **Dashboard**
   - Timetable view (calendar grid)
   - Faculty list with availability
   - Quick actions: Add leave, View substitutions

3. **Timetable Editor**
   - Drag-and-drop timetable builder
   - Conflict visualization
   - Room/shift selectors
   - Faculty/subject pickers

4. **Chat Interface**
   - Natural language chat box
   - Real-time substitution suggestions
   - Confirmation buttons
   - Faculty availability display

## Components
- **Shadcn UI Components**
  - Buttons (primary, secondary)
  - Cards with faculty/substitution info
  - Calendar grid with time slots
  - Dropdown selectors for subjects/rooms

- **Design Specifications**
  - Color palette: Primary #007bff, Secondary #6c757d, Background #f8f9fa
  - Typography: Arial, 14px base size
  - Spacing: 8px grid system
  - Responsive breakpoints: Mobile, Tablet, Desktop
