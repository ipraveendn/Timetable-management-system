# VYUHA - AI-Powered Timetable & Substitution System

VYUHA is a state-of-the-art SaaS platform designed for modern colleges to automate timetable generation and faculty substitution.

## 🚀 Features
- **Automated Timetable Generation:** Conflict-free scheduling with 6-rule validation.
- **AI-Powered Substitution:** Instant faculty replacement based on expertise and load.
- **Centralized Portals:** Dedicated dashboards for Admins and Teachers.
- **Secure Infrastructure:** Powered by FastAPI, React, and Supabase.

## 📁 Project Structure
- **/frontend**: React + Vite application.
- **/backend**: FastAPI + Python application.
- **/docs**: Detailed documentation and implementation guides.
- **/scripts**: Database and maintenance utility scripts.
